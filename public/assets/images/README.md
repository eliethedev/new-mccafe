# Payment QR Codes

This directory should contain the following QR code images for payment processing:

- `gcash-qr.png` - GCash QR code for payments
- `paymaya-qr.png` - PayMaya QR code for payments

## Instructions

1. Generate actual QR codes for your GCash and PayMaya accounts
2. Save them as PNG files with the names specified above
3. Place them in this directory

## Temporary Placeholders

For development purposes, you can use placeholder images or leave these files missing. The payment pages will still function, but QR codes won't be displayed.
